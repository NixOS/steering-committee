# 0020: Move nixpkgs-update to the NixOS org and fund its servers

## Context and Subject
`nixpkgs-update` is the automated tooling that files version-bump pull requests against Nixpkgs, historically run under the nix-community umbrella. The Steering Committee considers it one of the project's critical tools and proposed bringing it into the NixOS GitHub organization, with the NixOS Foundation covering its server costs.

**Proposed By:** JulienMalka

## Motion
> The SC welcomes the `nixpkgs-update` tooling into the NixOS GitHub organization and approves the NixOS Foundation paying for its server costs at cost, recognizing it as one of the project's critical tools that the NixOS org should own and support.
